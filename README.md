# EpiRun
EpiRun es un videojuego de género Endless runner orientado a dispositivos móviles con una estética pixel art en 2D y desplazamiento horizontal. 
El jugador debe controlar a la protagonista para salvar los obstáculos y conseguir los objetos que van apareciendo por el camino.

# Estado actual del proyecto:

En esta versión se ha implementado una base del juego que permite probar algunas de las mecánicas principales en un entorno de pruebas.

Las funcionalidades actuales incluyen:

- Movimiento automático del escenario (scroll horizontal)
- Control del personaje mediante salto y doble salto (Espacio)
- Obstáculos básicos con detección de colisión
- Sistema de pausa (tecla "Esc")
- Menú principal con navegación básica
- Botón de salir del juego


# Cómo ejecutar el juego (build)

1. Descargar el archivo ".zip" desde la sección Releases del repositorio.
2. Descomprimir el archivo en una carpeta local.
3. Ejecutar el archivo EpiRun.exe
