# EpiRun
EpiRun es un videojuego de género Endless runner orientado a dispositivos móviles con una estética pixel art en 2D y desplazamiento horizontal. 
El jugador debe controlar a la protagonista para salvar los obstáculos y conseguir los objetos que van apareciendo por el camino.
